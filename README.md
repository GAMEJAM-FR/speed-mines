# speed-mines
Minesweeper in ruby/gosu. Goal: Same as classical minesweeper but with osu-like gamaplay.

### Get started

You may install ruby and gosu library first.

### Execution
* on linux

  ```
    git clone https://github.com/gjsheep/speed-mines
    cd speed-mines
    ./sm.rb
  ```

* on windows

always boot to linux
